== InterstellaReddit

Was planned to be a Reddit clone for the interstellar community.

* Posts (not yet implemented)

* Comments/Replies (not yet implemented)

* Users

* User Authentication (not yet implemented)

* Planets (Every User has a home planet)

* Star Systems

* ...


Just run <tt>bundler</tt>, then run <tt>bin/rails server</tt> and you should be good to go.
